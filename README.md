Bus Ticket Reservation System in C# 
````
Objective
-	To make the seat reservation online so that end user can book ticket from their own place
````
````
What is Needed
	User
  -	Login Screen involving username and password
  -	Reservation tab
    o	Source station to destination
    o	Date
    o	Button that loads available seats
    o	Enter, opens another screen that allows the user to enter first name and last name, and phone number
    o	Cancel option
    o	Ticket, can be saved to pdf
 Admin
  -	Login Screen involving username and password
  - Update calculation price 
  - Update bus status, cancelled, full etc.
  - Have access to user input data
  ````
